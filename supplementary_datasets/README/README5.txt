This supplementary dataset contains orthologous gene families predictions.

PredictedOrthoFamilies_Mouse_Rat_Chicken.txt: this file contains families of predicted 1-to-1 orthologous loci for mouse, rat and chicken, for all categories of genes. For each species, we provide the gene type and (for non-lncRNA loci) the reason why this locus was not included in the lncRNA dataset.

StatsProjections_From${species1}_To${species2}.txt: these files contain statistics for the exon projection and filtering procedure, from one species to another.

EnsemblOrtho_ProteinCodingGenes_1to1_${species_set}: these files contain orthology relationships extracted from the Ensembl 94 database, for protein-coding genes.

AlignmentStatistics_PredictedOrthoFamilies_${species1}_${species2}.txt: these files contain statistics (ungapped sequence length, number of identical bases) for cDNA alignments obtained with TBA and lastz (Methods).
