This supplementary dataset contains the results of the expression divergence analysis presented in Figure 9.

ExpressionDivergence_OrthoGenes_Mouse_Rat.txt: total expression divergence and percentage expression divergence contributed by each organ/developmental stage combination, for each pair of orthologus protein-coding genes and lncRNAs.

GOEnrichment_BiologicalProcess_Top10Percent_ProteinCodingGenes.txt: gene ontology enrichment for most divergent (top 10%) protein-coding genes.
